# P4716 【模板】最小树形图

## 题目背景

这是一道模板题。

## 题目描述

给定包含 $n$ 个结点， $m$ 条有向边的一个图。试求一棵以结点 $r$ 为根的最小树形图，并输出最小树形图每条边的权值之和，如果没有以 $r$ 为根的最小树形图，输出 $-1$。

## 输入格式

第一行包含三个整数 $n,m,r$，意义同题目所述。

接下来 $m$ 行，每行包含三个整数 $u,v,w$，表示图中存在一条从 $u$ 指向 $v$ 的权值为 $w$ 的有向边。

## 输出格式

如果原图中存在以 $r$ 为根的最小树形图，就输出最小树形图每条边的权值之和，否则输出 $-1$。

### 输入 #1

```
4 6 1
1 2 3
1 3 1
4 1 2
4 2 2
3 2 1
3 4 1
```

### 输出 #1

```
3
```

## 输入输出样例 #2

### 输入 #2

```
4 6 3

1 3 1
4 1 2

3 4 1
```

### 输出 #2

```
4
```

## 输入输出样例 #3

### 输入 #3

```
4 6 2

4 1 2
4 2 2
3 2 1
3 4 1
```

### 输出 #3

```
-1
```

## 说明/提示

**样例 $1$ 解释**

最小树形图中包含第 $2$， $5$， $6$ 三条边，总权值为 $1 + 1 + 1 = 3$

**样例 $2$ 解释**

最小树形图中包含第 $3$， $5$， $6$ 三条边，总权值为 $2 + 1 + 1 = 4$

**样例 $3$ 解释**

无法构成最小树形图，故输出 $-1$ 。

**数据范围**

对于所有数据，$1 \leq u, v \leq n \leq 100$, $1 \leq m \leq 10^4$​​, $1 \leq w \leq 10^6$​​。


这版代码已经具备了朱刘算法完整的灵魂，**框架和思路都是 100% 正确的**！

但是，在将其转化为实际可运行的代码时，你留下了几个**“极其危险的陷阱”和笔误**。如果直接编译运行，会面临编译报错、死循环和数据溢出的问题。

我们一行一行来排雷，一共 **5 个 Bug**，改完就能直接去洛谷 AC 模板题了：

### Bug 1：编译报错（缺少默认构造函数）
```cpp
vector<Edge> edges(m);
```
因为你给 `Edge` 结构体写了一个带参数的构造函数 `Edge(int u,int v,int w)`，C++ 编译器会自动收回默认构造函数。当你想初始化一个大小为 `m` 的 `vector` 时，由于没有默认构造函数，编译器会报错。
**修复：** 在 `Edge` 结构体里补上一个空的默认构造函数。
```cpp
struct Edge{
    int u, v, w;
    Edge() {} // 补上这一句
    Edge(int u,int v,int w){
        this->u=u; this->v=v; this->w=w;
    }
};
```

### Bug 2：致命逻辑崩溃（第一轮 `n` 变成 0）
```cpp
    int cnt=0; 
    while(true){
        n=cnt; // <--- 灾难发生在这里！
        vector<pair<int,int>> min_in(n+1,make_pair(-1,INT_MAX));
```
在第一轮循环刚开始时，`cnt` 是 `0`。你直接执行了 `n = cnt`，导致总节点数 `n` 变成了 0！接下来分配数组大小全部变成了 1，图完全消失了。
**修复：** `n = cnt;` 必须放在一轮完全结束、准备进入下一轮的时候。
把 `n = cnt;` **删掉**，放到 `while` 循环的**最后一行**（紧跟在 `edges[i] = Edge(u2,v2,w);` 循环的后面）。

### Bug 3：数据溢出（先加 `INT_MAX` 再判断）
```cpp
        for(int i=1;i<=n;i++){   //计算cost
            if(i!=r){
                cost+=min_in[i].second; // 如果是 INT_MAX，这里 cost 就溢出了！
            }
            if(i != r && min_in[i].second == INT_MAX) { ... }
        }
```
你应该**先判断**是不是孤立点，**再累加代价**。否则 `INT_MAX` 加到一个普通 `int` 上会导致数值溢出变成负数。
**修复：**
```cpp
        for(int i=1;i<=n;i++){   
            if(i == r) continue;
            if(min_in[i].second == INT_MAX) {
                cout << -1 << endl;
                return 0;
            }
            cost += min_in[i].second; // 确保安全了再加
        }
```

### Bug 4：低级笔误（自己赋给自己）
```cpp
            int u2=loop_id[u1];
            int v2=loop_id[v2]; // <--- 这里的 v2 写错了！
```
你试图用还没初始化的 `v2` 作为数组下标，这会导致严重的越界错误。
**修复：** `int v2 = loop_id[v1];`

### Bug 5：隐藏炸弹（`parent` 数组状态残留）
在 `while(true)` 循环中，由于每一轮图的结构都在变化，`parent` 数组其实是需要重置的。虽然你用 `if` 更新了 `parent`，但如果你不重置，上一轮的脏数据可能会残留。
**更优雅的修复（极简技巧）：**
你其实根本不需要全局维护一个 `parent` 数组！你在找最小入边时，`min_in[v].first` 记录的刚好就是指向 `v` 的点（即 parent）。
所以：
1. 直接**删掉**最外面定义的 `vector<int> parent(n+1,-1);`。
2. 在找最小入边时，只更新 `min_in` 即可。
3. 在下面找环的时候，把所有的 `v = parent[v];` 替换成 `v = min_in[v].first;`。
这不仅修复了炸弹，还让代码少了一个多余的数组！

### 最后确认：编号范围（洛谷 vs 你的题目）
这段代码使用的是 **`1` 到 `n`** 编号逻辑。
*   **对于洛谷 P4716 模板题**：题目给定节点是 1 到 n，所以这段代码跑洛谷是 **完美无瑕** 的（只要修复了上述 Bug）。
*   **对于你最开始发的那道原题**：题目特别说明 *"Note that the people are numbered from 0 to n−1"*。如果去跑那道题，你需要把所有的 `for (int i=1; i<=n; i++)` 改成 `0 到 n-1`。

### 修改后的 `while` 核心框架（参考）
把这几处改掉后，你的算法逻辑就成型了：

```cpp
    while(true){
        vector<pair<int,int>> min_in(n+1,make_pair(-1,INT_MAX)); 
        for(int i=0;i<m;i++){  
            int u=edges[i].u, v=edges[i].v, w=edges[i].w;
            if(v!=r && u!=v && min_in[v].second>w){
                min_in[v].first=u;
                min_in[v].second=w;
            }
        }
        
        for(int i=1;i<=n;i++){   
            if(i == r) continue;
            if(min_in[i].second == INT_MAX) {
                cout << -1 << endl; 
                return 0;
            }
            cost += min_in[i].second;
        }

        vector<int> visited(n+1,0); 
        vector<int> loop_id(n+1,0); 
        cnt=0;
        bool has_loop=false;
        for(int i=1;i<=n;i++){  
            int v=i;
            while(true){
                if(v==r) break;
                else if(loop_id[v]!=0) break;
                else if(visited[v]==i){
                    has_loop=true;
                    int origin=v;
                    cnt++;
                    loop_id[v]=cnt;
                    v=min_in[v].first; // 使用 min_in 直接作为 parent
                    while(v!=origin){
                        loop_id[v]=cnt;
                        v=min_in[v].first;
                    }
                    break;
                }
                visited[v]=i;
                v=min_in[v].first;
            }
        }
        
        if(!has_loop){
            cout << cost << endl;
            break;
        }
        
        for(int i=0;i<m;i++) edges[i].w -= min_in[edges[i].v].second;
        
        for(int i=1;i<=n;i++){
            if(loop_id[i]==0) loop_id[i] = ++cnt;
        }
        r = loop_id[r];
        
        for(int i=0;i<m;i++){
            int u1=edges[i].u, v1=edges[i].v, w=edges[i].w;
            edges[i] = Edge(loop_id[u1], loop_id[v1], w);
        }
        n = cnt; // 在这里更新 n ！
    }
```
